Para correr el código necesitas tener Python instalado, en caso de no tenerla, ingresar a la pagina https://www.python.org/ y descargarlo.

Paso uno, descomprimir el .rar y guardarlo en la carpeta.

Paso dos, abrir visual studio code y abrir la carpeta.

Paso tres, descargar las extensiones Python y Python debugger.

paso cuatro, abrir el archivo optimización.py y ejecutarlo (para ejecutarlo darle click al símbolo de "play" que se encuentra en la barra superior en la parte derecha).
